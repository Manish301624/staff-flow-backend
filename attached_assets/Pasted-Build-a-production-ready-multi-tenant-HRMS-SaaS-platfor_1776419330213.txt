Build a production-ready multi-tenant HRMS SaaS platform named "StaffFlow" with complete authentication (Admin + Manager + Employee) and full payroll + attendance system.

-----------------------------------
🔷 TECH STACK:

Frontend:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

Backend:
- Node.js (NestJS)

Database:
- PostgreSQL

Auth:
- JWT + OTP + optional PIN

-----------------------------------
🔷 USER ROLES:

- Admin
- Manager
- Employee

-----------------------------------
🔷 AUTHENTICATION SYSTEM (IMPORTANT)

1. ADMIN LOGIN:
- Email + Password
- JWT authentication

-----------------------------------
2. EMPLOYEE / MANAGER LOGIN:

STEP 1: Enter Mobile Number  
STEP 2: Send OTP (4-digit)  
STEP 3: Verify OTP  
STEP 4 (optional): Set 4-digit PIN  

-----------------------------------
🔷 OTP APIs:

POST /auth/send-otp
- Input: phone
- Generate 4-digit OTP
- Store with expiry (2 min)
- Return success

POST /auth/verify-otp
- Input: phone, otp
- Validate OTP
- If valid:
   - Find user in DB
   - If not found → create user (default role = employee)
   - Return JWT token + user

🔷 PAYROLL ENGINE:

- Fetch attendance
- Calculate present days
- Calculate payable days
- Compute salary:

per_day_salary = monthly_salary / total_days
base_salary = per_day_salary * payable_days

Add:
- bonus
- overtime

Deduct:
- PF, ESI, PT, TDS
- loan, advance

Final:
net_salary = base + earnings - deductions
